---------------------------------Intelligent Collection Shopping---------------------------------
Built using -
react native
node.js

Required packages -
react-native-router-flux
express 
mongoose 
body-parser 
connect-flash 
dotenv 
express-validator

--------------------------------------------------------------
To create a new project -
react-native init IntelligentCollectionShopping
cd IntelligentCollectionShopping
react-native run-android
npm uninstall react-native
npm install --save react-native@0.55.4
react-native run-android
npm install --save babel-core@latest babel-loader@latest
npm uninstall --save babel-preset-react-native
npm install --save babel-preset-react-native@4.0.0
react-native run-android
----------------------------------------------------------
Inside ICS_client folder -
npm install --save react-native-router-flux
npm install express mongoose body-parser connect-flash dotenv express-validator

Inside ISC_server folder-
npm i nodemon -g
yarn
-----------------------------------------------------------
To run -
node server.js
react-native run-android
----------------------------------------------------------
